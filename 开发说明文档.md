## 第一部分:项目背景

### 1.1 用户画像

- **学生**: 5 年级在读,DP(Dependent Pass)身份
- **PSLE 时间**: 2027 年 9 月(73 周备考期)
- **当前成绩** (P5):
  - 英语 65 分(AL5/6,最大短板)
  - 科学 85 分(AL2,中等偏上)
  - 数学 90+ 分(AL1)
  - 华文 90+ 分(AL1)
- **英语失分点**: 阅读 + 完形 + 作文 + 口语(整体语感不足)
- **抗压能力**: 一般,需要劳逸结合
- **特长 / DSA**: 无,纯靠笔试
- **PSLE 目标**: 总分 9-12 分(DP 良好中档校,如立化/英华/淡马锡)
- **作息要求**:
  - 平日 21:30 准时睡觉
  - 周日下午 14:00-18:00 完全休息(雷打不动)
  - 周六 19:30 后 2h 自由家庭活动

### 1.2 备考手册总览(配套文档)

学生使用的备考手册有两份:
- **PSLE_2027_完整备考总手册_v9.pdf**(79 页)— 包含 73 周完整周卡片
- **PSLE_2027_积分激励系统.pdf**(2 页 A4)— 积分规则单页

**手册核心结构**:
```
四阶段 73 周计划:
├── 第一阶段 W1-W26 (2026.5-10) "基础补漏"
│   ├── W1-W14: P3-P4 系统过 + 难易分级
│   └── W15-W26: P5 收尾
├── 第二阶段 W27-W52 (2026.11-2027.4) "系统提升"
│   ├── W27-W42: P6 系统学
│   └── W43-W52: 真题前移
├── 第三阶段 W53-W65 (2027.5-7) "刷题冲刺"
└── 第四阶段 W66-W74 (2027.8-9) "考前调整"
```

**每周时间分配 30.25 小时**:
- 英语 13.75h (45%) - 最大投入,因为最弱
- 科学 8.25h
- 数学 3.5h
- 华文 2.75h
- 复盘 2h

**每日时段模板**:
```
平日:
  16:30-18:00 段 1 学习 (1.5h)
  19:00-20:30 段 2 学习 (1.5h)
  20:30-21:00 段 3 学习 (0.5h)
  21:30 睡觉

周六: 7.5h 学习 (上午科学 + 下午英语到 19:30)
周日: 6h 学习 (上午科学华文 + 下午休息 + 晚上数学)
```

## 第二部分:积分激励系统(核心业务规则)

### 2.1 积分规则(必须严格遵守)

#### 加分项

| 类型 | 触发条件 | 积分 | 频率 |
|------|---------|------|------|
| 基础 | 每周日完成复盘 | +5 | 周 |
| 基础 | 月小测达标(W4 / W8 / W12) | +20 | 月 |
| 里程碑 | W14 中期模拟卷 (P3-P4 综合) | +50 | 单次 |
| 里程碑 | W20 P5 综合复习 | +30 | 单次 |
| **终极** | **W26 第一阶段总模考** | **+100** | **单次** |
| 额外 | 英语作文单篇 30+ 分 | +5 | 单次 |
| 额外 | 老师特别表扬(任一老师) | +10 | 单次 |
| 额外 | 连续 4 周无老师反馈学习问题 | +15 | 滚动 |

#### 扣分项

| 触发条件 | 扣分 |
|---------|------|
| 老师反馈学习问题(每次) | -10 |

**扣分细节**:
- 同周同老师只扣一次(避免轰炸)
- 不同老师独立计算
- 积分可扣到负数,但负数期间不能兑换
- 触发后角色 24 小时显示沮丧表情

#### "学习问题"定义(避免后期模糊)

✅ **算**:不交作业 / 课堂走神 / 错题不订正 / 作业敷衍 / 不专心听讲
❌ **不算**:某次发挥失常、某题不会(这是能力,不是态度)

### 2.2 兑换规则

- **汇率**: 1 分 = SGD 0.5
- **兑换方式**: 父母带孩子去现场买
- **❌ 不能买**: 游戏类(Switch 游戏卡 / Roblox 充值 / PS5 / 手游充值 / 电脑游戏 / 相关配件)
- **✅ 可以买**: 机器人、LEGO、玩具、文具、书、零食、衣服 — 其他都行
- **频率**: 每月可花一次,也可以攒着不花
- **有效期**: 无限期累积(可以攒到 PSLE 后)

### 2.3 第一阶段 26 周积分预估

| 时间 | 累积积分(正常发挥) | 兑换价值 SGD |
|------|---------------------|------------|
| W4 月评估 | ~45 | 22.5 |
| W8 | ~95 | 47.5 |
| W14 中期里程碑 | ~165 | 82.5 |
| W26 终极 | ~280 | 140 |

**17 个月全程预估**: ~740 分 = SGD 370

## 第三部分:技术架构

### 3.1 当前实现(v0.1)

**技术栈**:
- 纯前端: HTML + CSS + JavaScript(无框架)
- 数据存储: localStorage(浏览器本地)
- 单文件部署: 一个 `index.html`(63 KB)

**为什么选这个栈**:
- 用户需求是"GitHub Pages 部署"和"Firebase 云同步"
- 纯前端 = 无需服务器
- 单文件 = 部署/分享/备份都简单
- 无框架 = 任何浏览器都能跑

**文件结构**(开发时拆开,部署时合并):
```
chamui-psle/
├── index.html              # UI 骨架 + CSS
├── character.js            # 锤锤 SVG 渲染器
├── data.js                 # 数据存储 + 规则引擎
├── app.js                  # 主应用逻辑
└── chamui_app_single.html  # 合并后的单文件部署版
```

### 6.4 升级到 Firebase 云同步

参考 `锤锤备考_部署指南.md`,核心是替换 `loadState` / `saveState`:

```javascript
// 替换 data.js 中的两个函数

async function loadState() {
  return new Promise(resolve => {
    waitForFirebase(async () => {
      firebase.initializeApp(firebaseConfig);
      const db = firebase.firestore();
      const doc = await db.collection('chamui').doc('main').get();
      if (doc.exists) {
        resolve(Object.assign(getDefaultState(), doc.data()));
      } else {
        resolve(getDefaultState());
      }
    });
  });
}

async function saveState(state) {
  localStorage.setItem(STORAGE_KEY, JSON.stringify(state));  // 本地备份
  if (typeof firebase !== 'undefined') {
    await firebase.firestore().collection('chamui').doc('main').set(state);
  }
}
```

**注意**: `loadState` 改为 async 后,`init()` 也要改:

```javascript
async function init() {
  state = await loadState();
  recalcTotalPoints(state);
  bindEvents();
  renderAll();
}
```

3. **扣分**: 仅 -10 分一种,不能加其他扣分项
4. **总积分上限**: 第一阶段满分 ~370,正常 ~280
5. **里程碑周**: W4 / W8 / W12 / W14 / W20 / W26(对应备考手册)
6. **不能买游戏类**(包括 Switch / Roblox / PS5 等所有游戏 / 充值)
7. **可扣到负数**,但负数期间不能兑换

### 7.2 UI/UX 约束

1. **角色必须是原创**(避免抄袭锁哥锤等版权角色)
2. **沮丧表情持续 24 小时**,不能更长(避免太严厉打击孩子)
3. **不能因为扣分脱掉装备**(已解锁的装备永久保留)
4. **升级动画必须存在**(主要激励手段)
5. **关键周必须高亮**(W4/W8/W12/W14/W20/W26)

### 7.3 数据约束

1. **状态结构必须包含 version 字段**(为后续数据迁移)
2. **logs 必须包含 timestamp**(用于排序和"4 周滚动"计算)
3. **localStorage key**: `'chamui_psle_v1'`(改了会丢老数据)
4. **数据导出格式**: JSON,文件名 `chamui_psle_YYYY-MM-DD.json`

### 8.1 本地开发

```bash
# 直接打开
open chamui_app_single.html

# 或起本地服务器(支持 file:// 协议下不能用的功能)
python3 -m http.server 8000
```

**开发时拆分文件**:
- 编辑 `index.html` / `character.js` / `data.js` / `app.js`
- 用合并脚本生成 `chamui_app_single.html`(部署用)

**合并脚本** (build.py 或类似):
```python
with open('index.html') as f: html = f.read()
with open('character.js') as f: char = f.read()
with open('data.js') as f: data = f.read()
with open('app.js') as f: app = f.read()

merged = html.replace(
    '<script src="character.js"></script>\n<script src="data.js"></script>\n<script src="app.js"></script>',
    f'<script>\n{char}\n\n{data}\n\n{app}\n</script>'
)
with open('chamui_app_single.html', 'w') as f: f.write(merged)
```
